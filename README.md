# Borc-Hesaplama

İlama Bağlı Borçlar İçin Hesaplama

Avukatlar ve ilama bağlı borç ödeyen şirket ve kurumlar için borçların hesaplanmasında kullanılabilecek bir araçtır. Hesapların doğruluğu güncel verilerle mutlaka kontrol edilmelidir.
